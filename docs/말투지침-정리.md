## 1. 본문 공통 말투 규칙

### 핵심 원칙
- 자연스러운 문체로 작성할 것

## 6. 전체 금지 사항 요약

### 본문 금지
- 수면/불면 관련 언급
- 마크다운 강조 (**, *, #, `)
- 본문에 제품명(한려담원, 흑염소진액) 언급
- 짧은 문장 반복 패턴
- 직접적 자기소개 ("저는 40대 직장인입니다")
- 광고 티 나는 표현

### 댓글 금지
- 광고성 문구: "강추", "꼭 드세요", "무조건", "대박", "최고", "완전"
- 과장 표현: "완전히 달라졌어요", "인생이 바뀌었어요"
- 의료적 효능 단정: "~에 좋아요", "~가 치료돼요"
- 이모지 사용
- 반말 (존댓말 필수)
- "저도", "저는" 시작 댓글 연속
- 공감 댓글 2개 연속
- [댓글1]·[댓글2]에 제품 추천

### 작성자 답댓글 금지
- "감사합니다 먹어볼게요" 식 순수 긍정만 — 반드시 의심/우려/조건 질문 포함

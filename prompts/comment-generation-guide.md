# 댓글 생성 가이드 v2 (Codex 피드백 반영)

## 핵심 원칙
"네이버 카페 회원이 모바일로 5~20초 안에 쓸 법한 댓글"만 생성.
AI가 맥락을 정리해서 쓴 미니 후기가 아니라, 사람이 스크롤하다 툭 남기는 댓글.

## 카페 성격 정의 (댓글 생성 전 먼저 설정)
- 건강/생활/육아 계열 네이버 카페
- 회원 연령대: 25~55세 여성 위주 (남성 20% 정도)
- 존댓말 비율: 80% 이상
- 갈등 허용도: 낮음 (조심스러운 회의만, 정면충돌 금지)

## 댓글 역할 비율 (필수)
- 공감/응원: 40% — "축하드려요", "정보 감사해요", "저도 해봐야겠어요"
- 경험 공유: 25% — "저도 비슷했는데~", "저희 엄마도~"
- 정보 질문: 20% — "혹시 어디서?", "얼마예요?", "어떤 거 드세요?"
- 가벼운 추임새: 10% — "저장해둘게요", "좋은 정보네요"
- 조심스러운 회의: 5% — "개인차가 있을 것 같긴 한데", "혹시 부작용은 없으셨나요?"

## 길이 분포 (필수)
- 1문장 (70%): 대부분의 댓글
- 2문장 (25%): 질문+이유, 공감+경험
- 3문장 이상 (5%): 본인 경험 상세 공유

## 말투 규칙
★ 기본은 부드러운 존댓말/반존대 (카페 톤)
★ 금지 표현: "거임", "ㄹㅇ", "장땡", "놀람", "ㄷㄷ" 등 커뮤 밈체
★ 가족 호칭: "저희 엄마", "어머니", "따님" (존칭 우선. "우리 엄마", "딸내미" 지양)
★ 완충 장치: ㅎㅎ/ㅠㅠ/~/이모지 댓글당 최대 1개
★ ~요/~에요 3연속 금지
★ "맞아요" 시작 최대 2개

## 대댓글 스레드 규칙 (★ 매우 중요)
- ★ 질문 댓글에는 반드시 [작성자-N] 답글이 있어야 함
- ★ 전체의 30~40%가 대댓글 ([작성자-N]/[댓글러-N]/[제3자-N])
- 스레드 예:
  [댓글3] 질문
  [작성자-3] 답변
  [댓글러-3] 후속 반응
- 각 댓글은 독립 발화. 대화가 제품 추천으로 수렴 금지
- "광고 의심 → 해명 → 제3자 방어" 3단 콤보 금지
- 작성자 답글: 브랜드 유도/쪽지 유도 금지

## 출력 형식 (★ 필수)
- ★ 카테고리 헤더 금지. "[공감/응원 — 10개]" 같은 분류 절대 출력하지 말 것
- ★ 순서대로 [댓글1]부터 쭉 나열. 중간에 구분선/제목/빈줄 없이
- 각 줄이 [댓글N], [작성자-N], [댓글러-N], [제3자-N] 중 하나로 시작
- 볼드(**) 금지

## 문체 현실감
- ★ 너무 깔끔하게 정돈된 문장 금지. 실제 모바일 댓글은 더 어수선함
- 띄어쓰기 가끔 틀려도 OK ("저도그랬어요", "진짜좋은정보")
- 문장 끝이 매끈하게 마무리 안 돼도 OK ("이거 진짜.. 저도 해봐야하나")
- 생각 도중에 끊기는 댓글 OK ("아 근데 이거 어디서")

## 쪽지/브랜드 관련
★ 댓글에서 "쪽지 주시면 알려드릴게요" 금지 — 광고성 냄새
★ 브랜드 질문이 나오면 작성자는 "본문에 적어둔 게 전부예요" 또는 "댓글로 적어드릴게요" 정도
★ 광고 방어용 제3자 댓글 금지 ("글 톤 보면 광고 같진 않아 보여요" 이런 거)

## 의심/회의 댓글 규칙
- 전체의 5% 이하
- 정면충돌 금지 ("광고 아닌가요?" 금지)
- 완충 표현 필수: "혹시", "실례 안 되면", "개인차가 있겠지만"
- 예: "개인차가 있을 것 같긴 한데 혹시 부작용 같은 건 없으셨나요?"

## 본문 연결
★ 각 댓글은 본문의 특정 디테일을 최소 1개 직접 언급해야 함
★ 공허한 "공감돼요" 금지 — 본문의 어떤 부분에 공감하는지 구체적으로
